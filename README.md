# My Knowledge Base

Reusable engineering knowledge for Codex-assisted work.

This repository is organized as a knowledge harness:

- `docs/**`: SSOT documents.
- `indexes/routing-index.md`: choose which SSOT documents to read for a task.
- `indexes/summary-index.md`: compact overview for session injection.
- `kb.manifest.json`: machine-readable catalog.
- `AGENTS.md`: instructions for agents using this repository.

The core rule is: inject little, route precisely, read SSOT only when needed.
